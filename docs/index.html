Fontを利用した画像の表示

いんとろだくしょん

リソースパックを利用してFontに任意の画像を割り当てることで，テキストを表示できる画面に画像を表示できる。

例:GUIの背景や任意のスロット(GUIのタイトルから)，プレイヤーの体力ゲージ上部(Actionbarから)

利点

  • 文字ごとに色を自由に変更可能
  • 表示位置もほぼ自由に変更可能

欠点

  • アニメーションテクスチャは使用不可(shaderを利用したアニメーションは可能らしい)
  • GUIタイトルを変更したい場合，GUI自体を開き直す必要がある
  • 画面サイズやGUIの大きさによって，表示位置を定められない場合がある
  • Fontが表示される側
  • b
  • c

Fontが表示される側

文字が表示される時，文字は前の文字の後ろから1px間隔をあけて表示される。

例えば5pxの文字が二つ並べられた文字列，の長さは11px(5px+1px+5px)になる。

この1pxの間隔を変更するには，横幅が指定された空白文字を文字間に挿入する。
