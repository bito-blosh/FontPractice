Fontを利用した画像の表示

いんとろだくしょん

リソースパックを利用してFontに任意の画像を割り当てることで，テキストを表示できる場所に画像を表示できる。

例:GUIの背景や任意のスロット(GUIのタイトルから)，プレイヤーの体力ゲージ上部(Actionbarから)

メリット

  • 文字ごとに色を自由に変更可能
  • 表示位置もほぼ自由に変更可能

デメリット

  • アニメーションテクスチャは使用不可(shaderを利用したアニメーションは可能らしい)
  • GUIタイトルを変更したい場合，GUI自体を開き直す必要がある
  • 画面サイズやGUIの大きさによって，表示位置を定められない場合がある
